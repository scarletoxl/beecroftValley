## About

Beecroft Valley is inspired by games like Stardew Valley, bringing the peaceful lifestyle of Beecroft to life. Based on the **real map and landmarks** of Beecroft, NSW, this game lets you explore an authentic recreation of this leafy Sydney suburb. Tend to your garden, explore local landmarks, chat with friendly neighbors, and experience the Beecroft community in this relaxing browser-based game!

## Controls

- **Arrow Keys** or **WASD** - Move your character
- **Space** - Talk to NPCs and interact with buildings
- **E** - Use currently selected tool
- **Mouse Click** - Select tools from toolbar

## Getting Started

Simply open `index.html` in your web browser to start playing!

For development with live server:
```bash
npm start
# Game will be available at http://localhost:8000
```

## Technology Stack

- HTML5 Canvas for rendering with camera scrolling
- Vanilla JavaScript for game engine
- CSS3 for UI styling
- Procedurally generated tree placement
- No build process required!
